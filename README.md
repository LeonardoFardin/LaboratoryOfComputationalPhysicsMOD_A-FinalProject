# Laboratory of Computational Physics MOD A — Final Project

Final group project developed with G. Bregolin, L. Fogarolo, and F. Peron, awarded 30 cum laude.

## Overview
This project explores the Iterated Prisoner's Dilemma from a computational perspective. 
We implemented and analyzed different strategies to better understand their behavior over repeated interactions.

The main goal of this project is to showcase the Python programming skills developed during the LCP MOD-A course. 
It is not intended to provide a fully optimized simulation or an exhaustive analysis of the problem.

## Contents
- `notebook.ipynb`: main Jupyter notebook containing the analysis and simulations
- `.pkl` files: precomputed data used in sections 3 and 4

The provided data files are included because generating them requires significant computational time.

## How to run
Install the required dependencies and run the notebook using Jupyter.
